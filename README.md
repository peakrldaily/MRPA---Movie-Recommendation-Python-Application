# MRPA---Movie-Recommendation-Python-Application

**1. INSTALL PYTHON:**
  https://www.python.org/downloads/
  
**2. RUN THIS COMMAND IN TERMINAL:**
```
pip install requests PySide6 Pillow
```
**3. ONCE PYTHON FILE IS RUNNING, INPUT YOUR API KEYS INTO THE SETTINGS PAGE**

  YOU CAN GET YOUR API KEYS FROM THESE LINKS:
  
    GEMINI - https://aistudio.google.com/
    OMDB - https://www.omdbapi.com/apikey.aspx
**4. NOW ASK FOR SUGGESTIONS!**
